# Fab.Person

`Fab.Person` is an Elixir library for generating random information about a person.

## Installation

Add `:fab_person` to your list of dependencies in `mix.exs`:

```elixir
def deps do
  [
    {:fab_person, "~> 1.0"}
  ]
end
```
